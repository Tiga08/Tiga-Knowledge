# 02-ai

AI、CV、LLM、模型部署与推理优化。

## 子目录

| 目录 | 说明 |
|------|------|
| `computer-vision/` | 图像分类、检测、分割、OCR、跟踪等 |
| `deep-learning/` | 深度学习基础、网络结构、训练方法 |
| `llm/` | 大语言模型原理与应用 |
| `multimodal/` | 多模态模型 |
| `agent/` | Agent 原理、框架与应用 |
| `model-deployment/` | ONNX、TensorRT、OpenVINO、服务化部署 |
| `model-compression/` | 剪枝、量化、蒸馏、轻量化 |
| `inference-acceleration/` | CUDA、Triton、CPU/GPU 推理优化 |
| `data-engineering/` | 数据清洗、标注、增强、数据集管理 |
