# 05-papers

论文阅读、论文笔记、复现记录。

## 推荐分类（按需创建）

`computer-vision` · `llm` · `multimodal` · `agent` · `model-compression` · `inference-acceleration` · `engineering`

## 单篇论文目录结构

每篇论文一个目录，命名 `year-paper-short-name/`（如 `2023-segment-anything/`）：

```text
year-paper-short-name/
├── README.md       # 论文信息与阅读状态
├── summary.zh.md   # 中文摘要
├── notes.md        # 阅读笔记
└── assets/         # 图片附件
```

可按需增加 `formulas.md`、`implementation.md`、`experiments.md`、`references.md`。笔记模板统一放在 `91-templates/`。

## 约定

不得编造实验结果、公式或作者观点。
